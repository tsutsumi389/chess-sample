# チェスゲーム作成指示書

## 概要

Babylon.js + TypeScript + Vite を使ってブラウザで動作するチェスゲームを作成してください。
ルール・ユニット・勝利条件はすべて標準チェスに準拠します。

---

## 技術スタック

| 項目 | 内容 |
|------|------|
| 3Dエンジン | Babylon.js（最新版） |
| 言語 | TypeScript |
| ビルドツール | Vite |
| 対象環境 | モダンブラウザ（Chrome / Firefox / Safari） |

---

## 機能要件

### 1. ボード

- 8×8の四角グリッド
- 白・黒の市松模様（標準チェスボード配色）
- マスをクリックしたとき、選択中の駒がある場合は移動先として処理する
- 選択中の駒の移動可能マスをハイライト表示する（半透明の色で示す）

### 2. カメラ

- **ArcRotateCamera**（Babylon.js標準）を使用
- 初期位置: ボードを斜め上から見下ろす角度（α: -π/4、β: π/3、radius: 20）
- マウスドラッグで回転、ホイールでズーム可能
- ズーム範囲: radius 8〜40

### 3. 駒

標準チェスの16種（白8・黒8）を初期配置する。

| 駒 | 英語名 | 初期配置（白） | 初期配置（黒） |
|----|--------|--------------|--------------|
| キング | King | e1 | e8 |
| クイーン | Queen | d1 | d8 |
| ルーク | Rook | a1, h1 | a8, h8 |
| ビショップ | Bishop | c1, f1 | c8, f8 |
| ナイト | Knight | b1, g1 | b8, g8 |
| ポーン | Pawn | a2〜h2 | a7〜h7 |

**駒の外見:**
- 3Dメッシュで表現する（シンプルな形状でOK）
  - キング: 背の高い円柱＋球
  - クイーン: 円柱＋先端に小さい球
  - ルーク: 四角柱
  - ビショップ: 細長い円柱
  - ナイト: 円柱＋斜めに傾いた球（L字的な形）
  - ポーン: 小さい円柱＋球
- 白駒: `#F0D9B5`（クリーム色）
- 黒駒: `#B58863`（茶色）
- 選択中の駒: 上方向に少し浮かせるアニメーション（Y軸 +0.3）

### 4. 移動ルール

標準チェスのルールをすべて実装する。

| 駒 | 移動ルール |
|----|-----------|
| キング | 上下左右斜め1マス。キャスリング対応 |
| クイーン | 上下左右斜め、障害物がなければ何マスでも |
| ルーク | 上下左右、障害物がなければ何マスでも |
| ビショップ | 斜め、障害物がなければ何マスでも |
| ナイト | L字移動（2+1）。駒を飛び越せる |
| ポーン | 前1マス（初手のみ2マス可）。斜め前に敵駒があれば取れる。アンパッサン対応 |

その他のルール:
- **チェック**: キングが攻撃されている状態を検出・表示する
- **チェックメイト**: キングが逃げられない状態で勝利判定
- **ステールメイト**: 合法手がない引き分け判定
- **プロモーション**: ポーンが最終段に達したらクイーンに昇格（自動）
- **キャスリング**: キング・ルークが初手未使用かつ間が空の場合に実行可能

### 5. ターン管理

- 白番からスタート
- 画面上部にどちらの番かを表示（「White's Turn」/ 「Black's Turn」）
- チェック状態のとき「Check!」と表示
- チェックメイト・ステールメイト時にゲーム終了メッセージを表示

### 6. UI

- 画面上部に現在のターン・状態を表示するテキストオーバーレイ（HTML/CSS）
- ゲーム終了時にリスタートボタンを表示
- 駒クリックで選択、選択中に別の駒をクリックで選択変更

---

## 非機能要件

- **AI対戦は不要**（2プレイヤーのローカル対戦のみ）
- **音声は不要**
- **セーブ・ロード機能は不要**
- コードは TypeScript の型を適切につける（`any` は極力使わない）
- ファイルは役割ごとに分割し、1ファイルに詰め込まない
